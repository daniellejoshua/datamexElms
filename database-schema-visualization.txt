    🔄 SYSTEM WORKFLOWS & BUSINESS PROCESSES:
    ═══════════════════════════════════════════════════════════════════

    👨‍💼 ADMIN/HEAD TEACHER WORKFLOW:
    ──────────────────────────────────────────────────────────────────
    1. 📝 INITIAL SYSTEM SETUP
       ├─ Create user accounts for teachers, registrar, students
       ├─ Define permanent course catalog (subject codes, units)
       ├─ Create sections for specific academic year/semester periods
       ├─ Assign rooms to sections
       └─ Configure class schedules (day, time, room)

    2. 👨‍🏫 TEACHER MANAGEMENT
       ├─ Assign teachers to specific sections
       ├─ Track teacher specializations and departments
       ├─ Monitor teaching loads and assignments
       └─ Manage teacher status and employment records

    3. 📢 ANNOUNCEMENT SYSTEM
       ├─ Create announcements with priority levels
       ├─ Set visibility (teachers_only, all_users, students_only)
       ├─ Attach files (documents, images, PDFs)
       ├─ Schedule publication and expiration dates
       ├─ Archive old announcements
       └─ Monitor read status and engagement

    4. 📊 ACADEMIC OVERSIGHT
       ├─ Monitor overall grade distributions
       ├─ Review academic standing reports
       ├─ Generate institutional analytics
       ├─ Track graduation requirements progress
       └─ Oversee transcript generation

    👩‍🏫 TEACHER WORKFLOW:
    ──────────────────────────────────────────────────────────────────
    1. 📚 COURSE MANAGEMENT
       ├─ View assigned sections and class schedules
       ├─ Access student rosters for each section
       ├─ Upload course materials by category:
       │  ├─ Lecture notes and presentations
       │  ├─ Assignments and projects
       │  ├─ Reading materials and references
       │  └─ Exam schedules and guidelines
       └─ Track material download statistics

    2. 📝 GRADING PROCESS (5-Component System)
       ├─ **PRELIM PERIOD** (Week 1-6):
       │  ├─ Record prelim grades for enrolled students
       │  ├─ Add teacher remarks and feedback
       │  └─ Submit prelim grades with timestamp
       │
       ├─ **MIDTERM PERIOD** (Week 7-12):
       │  ├─ Record midterm grades
       │  ├─ Review student progress from prelim
       │  └─ Submit midterm grades with timestamp
       │
       ├─ **PREFINALS PERIOD** (Week 13-16):
       │  ├─ Record prefinal grades
       │  ├─ Assess overall semester performance
       │  └─ Submit prefinal grades with timestamp
       │
       ├─ **FINALS PERIOD** (Week 17-18):
       │  ├─ Conduct final examinations
       │  ├─ Record final grades
       │  └─ Submit final grades with timestamp
       │
       └─ **SEMESTER GRADE CALCULATION**:
          ├─ System auto-calculates: (Prelim + Midterm + Prefinals + Finals) ÷ 4
          ├─ Teacher reviews and confirms semester grade
          ├─ Set overall completion status (Passed/Failed/Incomplete)
          └─ Final submission triggers transcript entry

    3. 👀 STUDENT MONITORING
       ├─ Track student attendance and participation
       ├─ Monitor payment status before grade submission
       ├─ Identify struggling students for intervention
       ├─ Generate progress reports per grading period
       └─ Communicate with parents/guardians as needed

    👩‍💼 REGISTRAR WORKFLOW:
    ──────────────────────────────────────────────────────────────────
    1. 🎓 STUDENT ENROLLMENT MANAGEMENT
       ├─ **REGULAR STUDENTS**:
       │  ├─ Enroll in single section per semester
       │  ├─ Verify payment status before enrollment
       │  ├─ Assign to appropriate year level section
       │  └─ Generate enrollment confirmation
       │
       ├─ **IRREGULAR STUDENTS**:
       │  ├─ Enroll in multiple sections (mixed year levels)
       │  ├─ Track subject completions and requirements
       │  ├─ Manage transfer credits and course equivalencies
       │  ├─ Calculate remaining units for graduation
